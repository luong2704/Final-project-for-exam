**Code-behind (CategoryFilterView.xaml.cs):**
```csharp
namespace Campus.Views;

public partial class CategoryFilterView : ContentPage
{
    public CategoryFilterView(CategoryViewModel viewModel)
    {
        InitializeComponent();
        BindingContext = viewModel;
    }

    protected override async void OnAppearing()
    {
        base.OnAppearing();
        if (BindingContext is CategoryViewModel vm && vm.Categories.Count == 0)
        {
            await vm.LoadCategoriesAsync();
        }
    }
}
```

Key requirements:
- Use Border (NOT Frame — Frame is obsolete per .NET 9)
- Picker uses ItemDisplayBinding="CategoryName" to show category names
- Selected category label shows visual feedback when IsFiltered is true
- Clear button hidden when no filter active
- Follow file-scoped namespace: namespace Campus.Views;
- Constructor receives CategoryViewModel via DI (will be registered in MauiProgram.cs later)
