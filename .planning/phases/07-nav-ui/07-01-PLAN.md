  <action>
Create the Converters directory and EmptyCollectionConverter.cs file.

Create IValueConverter that converts collection count to boolean for empty state visibility:
- Input: int count
- Output: bool true when count == 0 (empty), false otherwise

```csharp
using System.Globalization;
using Microsoft.Maui.Controls;

namespace Campus.Converters;

public class EmptyCollectionConverter : IValueConverter
{
    public object Convert(object? value, Type targetType, object? parameter, CultureInfo culture)
    {
        if (value is int count)
        {
            return count == 0;
        }
        return false;
    }

    public object ConvertBack(object? value, Type targetType, object? parameter, CultureInfo culture)
    {
        throw new NotImplementedException();
    }
}
```

The converter will be used in CategoryFilterView.xaml to show/hide empty state label based on FilteredEvents.Count.

Add resource dictionary inside ContentPage.Resources:

```xml
</ContentPage.Resources>
```

**Step 2: Update empty state visibility**
The CollectionView already has `EmptyView="No events match your filter"` - this is the string empty view.

To make it dynamic with the converter, update the CollectionView empty view to a more prominent display. Replace the EmptyView string with a View inside EmptyView:

```xml
<CollectionView.ItemsLayout>
    <LinearItemsLayout Orientation="Vertical" />
</CollectionView.ItemsLayout>
<CollectionView.EmptyView>
    <VerticalStackLayout 
        HorizontalOptions="Center" 
        VerticalOptions="Center"
        Padding="30"
        IsVisible="{Binding FilteredEvents.Count, Converter={StaticResource EmptyCollectionConverter}}">
        <Label 
            Text="No events match your filter"
            FontSize="18"
            TextColor="#666666"
            HorizontalTextAlignment="Center"/>
        <Label 
            Text="Try selecting a different category"
            FontSize="14"
            TextColor="#999999"
            HorizontalTextAlignment="Center"
            Margin="0,10,0,0"/>
    </VerticalStackLayout>
</CollectionView.EmptyView>
```

Note: The CollectionView.EmptyView displays when ItemsSource is empty. If FilteredEvents starts with data but becomes empty after filter, the empty view will show automatically. The converter binding ensures consistent behavior.
